### 数据来源

- **股票行情**：baostock `query_history_k_data_plus()`，后复权，日度，2020-01-01 至今
- **市场指数**：baostock，沪深 300（000300，CAPM 市场基准）+ 中证 500（000905，中小盘代表，与沪深 300 互补，观察大小盘风格差异）
- **宏观指标**：
  - CPI 同比增速：akshare `macro_china_cpi_yearly()`，反映物价水平变化，与货币政策和股市估值密切相关
  - M2 同比增速：akshare `macro_china_money_supply()`，M2 衡量广义货币供应量，增速上升代表流动性宽松，理论上利好股市
- **财务数据**：baostock `query_profit_data()`，ROE（净资产收益率）和净利润率，10 只股票 × 5 年（2020-2024），长格式存储

### 存储方式

- **基础**：CSV（方式 A）— 所有原始数据和合并数据均以 CSV 格式存储
- **进阶**：Parquet（方式 B）— 清洗后数据额外保存一份 Parquet 格式
- **选择 Parquet 的理由**：Parquet 为列式存储，支持只读取需要的列，且内置数据类型契约（Schema），避免 CSV 的类型丢失问题。实测对比：Parquet 文件体积仅为 CSV 的 46.8%，读取速度快 60%

### CSV 格式优缺点

**优点**：纯文本，通用性强，人类可读，Git diff 友好。

**力不从心的场景**：数据量达 GB 级时读写慢、体积大；不保存数据类型（每次读入需重新指定 dtype）；不支持列式读取；多表关联查询不便。

### 如何运行

1. 克隆仓库：`git clone https://github.com/599625523-sketch/ex_p02.git`
2. 进入项目目录：`cd ex_p02/dshw-p01`
3. 安装依赖：`pip install -r requirements.txt`
4. 运行 `01_download.ipynb` 下载原始数据（需联网，baostock 需登录）
5. 运行 `02_clean.ipynb` 清洗并存储数据（生成 CSV + Parquet）
6. 运行 `03_analysis.ipynb` 查看描述统计、可视化与 CAPM 回归结果
7. 打开 `report.html` 阅读完整分析报告

### 说明

- `data/clean/` 和 `data/combined/` 中的清洗/合并后数据已上传，可直接运行 `03_analysis.ipynb`
